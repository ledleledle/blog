Cara membuat akun AWS Educate $100 Gratis

Dec 20, 2020

Hari ini saya akan bahas tentang AWS Server namun ini lebih ke pembuatan akunnya. Jadi AWS memang menyediakan layanan gratis untuk edukasi yang bernama AWS Educate. Disana, kita bisa mendapatkan akun AWS dengan credit senilai $100 dengan durasi 1 tahun. Nah ini cocok sekali untuk kalian yang ingin belajar DevOps/Infra (yang penting jangan buat minning 😂). Tentu saja blog ini akan terus membahas tentang cloud services, jadi akun AWS ini akan sangat membantu kalian dalam proses belajar.

Buat akun AWS Educate dulu. Pergi ke halaman AWS Educate, lalu klik menu Join AWS Educate.

Pilih role sebagai Murid/Student (Tergantung kebutuhan sih… Kalau disini saya memang sebagai murid). Lalu tolong isi form yang diberikan dengan benar. Karena email akan digunakan untuk konfirmasi nantinya. Kalau sekolah kalian tidak ada didalam list, cantumkan saja Kementerian Pendidikan dan Kebudayaan atau Kementerian Komunikasi dan Informatika.

Pastikan juga kalian memilih bulan dan tahun kelulusan (Graduation Month & Graduation Year) setahun dari sekarang, misal sekarang Desember 2020 pilih tanggal kelulusannya Desember 2021. Kira-kira tampilan akhirnya seperti ini.

Dah selanjutnya tinggal klik I Agree saja. Setelah pendaftaran selesai, maka kita akan mendapatkan email verifikasi dari AWS.

Setelah email sudah dikonfirmasi, saatnya menunggu email balasan dari AWS. Jika diterima seperti ini balasannya.

Sekarang coba login ke AWS Educate. Lalu pergi ke tab AWS Account. Klik AWS Educate Starter Account.

Nanti kita akan dialihkan ke website pihak ke-3. Disana kelihatan credits kalian tinggal berapa.

Nice kita sudah masuk pada page EC2 Management. Namun untuk praktek penggunaan EC2 ini tidak akan saya bahas sekarang.

Yeah… Met belajar cuyy!!!

Nantikan artikel berikutnya tentang penggunaan AWS Cloud. Sampai jumpa…!!!
