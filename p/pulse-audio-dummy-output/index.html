Pulse Audio Dummy Output

Mar 11, 2021
1 minute read

Dummy Output biasanya terjadi setelah update, umumnya ini terjadi pada OS yang memiliki karakteristik Rolling Release seperti Arch Linux. Sebenarnya kejadian ini sangat jarang terjadi, bahkan saya baru sekali ini mengalami kondisi seperti ini.

Solusi

1

pulseaudio --check; pulseaudio --kill; pulseaudio --start
