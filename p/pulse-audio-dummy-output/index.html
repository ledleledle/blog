Pulse Audio Dummy Output

Mar 11, 2021

Dummy Output biasanya terjadi setelah update, umumnya ini terjadi pada OS yang memiliki karakteristik Initial Release seperti Arch Linux. Sebenarnya kejadian ini sangat jarang terjadi, bahkan saya baru sekali ini mengalami kondisi seperti ini.

Solusi

pulseaudio --check; pulseaudio --kill; pulseaudio --start
