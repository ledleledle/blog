Siapakah saya

Halo… Nama saya Leon Prasetya Mulya, lahir di Depok dan besar di Trenggalek. Sedang mem-fokuskan diri pada bidang DevOps (karena sebenernya nggak suka ngoding). Menyukai Linux, UI/UX Design, Microservices dan masih banyak hal lain. Lulusan bootcamp Dumbways.id jurusan DevOps Batch 1 pada bulan November 2020. Pernah jadi DevOps selama 6 bulan. Dan sekarang saya bekerja sebagai Solution Architect (masih belajaran) disuatu perusahaan IT hehe.

Tujuan saya membuat Blog

Tujuan saya membuat blog ini adalah untuk catatan pribadi saya sendiri dan juga untuk sharing dengan teman-teman saya. Jadi jika ada penulisan yang tidak jelas atau ambigu, tolong dimaklumi dan dikoreksi/ditanyakan langsung. Dan sekarang saya belum ada pekerjaan tetap dan sangat banyak waktu luang, hal ini saya manfaatkan untuk belajar sembari menulis blog. Jujur saja, sebelumnya niatnya juga untuk pamer ke HRD dan Talent Acquisition biar saya cepat dapet kerja wkwkwk (it’s work like a magic)…
