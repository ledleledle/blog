Siapakah saya

Halo… Nama saya Leon Prasetya Mulya, lahir di Depok dan besar di Trenggalek. Sedang mem-fokuskan diri pada bidang DevOps. Menyukai Linux, UI/UX Design, Microservices dan masih banyak hal lain. Lulusan bootcamp Dumbways.id jurusan DevOps Batch 1 pada bulan November 2020.

Tujuan saya membuat Blog

Tujuan saya membuat blog ini adalah untuk catatan pribadi saya sendiri dan juga untuk sharing dengan teman-teman saya. Jadi jika ada penulisan yang tidak jelas atau ambigu, tolong dimaklumi dan dikoreksi/ditanyakan langsung. Dan sekarang saya belum ada pekerjaan tetap dan sangat banyak waktu luang, hal ini saya manfaatkan untuk belajar sembari menulis blog.

Jadi intinya, semakin saya sering belajar. Konten blog ini bakal bertambah banyak. Saya tidak akan memonetisasi blog ini, dan kalian bebas belajar disini (asal jangan main copas).
